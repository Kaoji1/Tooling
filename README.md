## Please read before doing project ##

## Introduction
This is web-application to request indirect expense from user and send the the request to the admin stock.
This web-application will reduce work load from labor.

## Preparing
check Node.js version
command: node -v

check npm version
command: npm -v

check Angular version
command: ng version

## Installing
if node doesn't exit install from https://nodejs.org/en 

Then verify installation
command: node -v
command: npm -v

Then install Angular
command: npm install -g @angular/cli@18

Then verify installation
command: ng version
